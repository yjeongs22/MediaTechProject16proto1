# 플랜픽 MVP v2

요청사항 반영 버전입니다.

## 반영된 내용

### 1. 선호도 1·2·3순위 제거
사용자는 자유 입력만 작성합니다.
앱이 자유 입력 내용을 바탕으로 우선순위를 자동 유추합니다.

예:
"오전 수업은 피하고 싶고 금공강이면 좋지만 졸업은 밀리면 안 돼요"
→ 졸업요건, 오전 수업 회피, 공강 순서로 유추

### 2. 시간표 강의 클릭 상세창
완성된 시간표에서 강의를 클릭하면 아래 정보를 볼 수 있습니다.

- 강의평
- 강의계획서
- 강의코드
- 강의실
- 학점
- 시간

### 3. 관리자 강의 데이터 수정
관리자 탭에서 현재 등록된 강의 정보를 직접 수정할 수 있습니다.

수정 가능한 정보:
- 과목명
- 구분
- 요일
- 시작/종료 시간
- 학점
- 강의코드
- 강의실
- 강의평
- 강의계획서
- 난이도
- 팀플
- 색상

### 4. 웹 UI로 변경
기존 모바일 앱 느낌에서 웹 대시보드 느낌으로 변경했습니다.

## 실행 방법

1. 압축을 풉니다.
2. `플랜픽 실행.bat` 파일을 더블클릭합니다.
3. 브라우저에서 앱이 실행됩니다.

또는 `index.html` 파일을 직접 더블클릭해도 됩니다.

## 빠른 시연 흐름

1. 홈 → 추천 요청 시작
2. 학교/학과/학번/학년 입력
3. 자유 입력 작성
4. 분석하기로 우선순위 확인
5. AI 분석 요청
6. 관리자 페이지
7. 요청 상세 보기
8. 예시 결과 자동 입력
9. 결과 저장하고 시간표 생성하기
10. 결과 화면에서 시간표 확인
11. 시간표 강의 클릭 → 상세창 확인

## 주의

- 이 버전은 HTML/CSS/JavaScript만 사용합니다.
- 데이터는 브라우저 localStorage에 저장됩니다.
- 같은 브라우저에서 사용자와 관리자 흐름을 시연하는 MVP입니다.
- 실제 여러 사용자가 접속하려면 Firebase 또는 Supabase 연결이 필요합니다.


## 다른 PC 관리자 사용 설정

무료 Firebase Firestore를 연결하면 사용자 PC에서 보낸 요청을 다른 PC의 관리자 화면에서 볼 수 있습니다.

1. Firebase 콘솔에서 무료 Spark 플랜 프로젝트를 만듭니다.
2. 웹 앱을 추가하고 Firebase config 값을 복사합니다.
3. `firebase-config.js`의 빈 값에 config 값을 붙여넣습니다.
4. Firestore Database를 만들고 테스트용 규칙으로 읽기/쓰기를 허용합니다.
5. 같은 폴더를 다른 PC에서 열거나, Firebase Hosting/GitHub Pages 같은 무료 호스팅에 올려 같은 주소로 접속합니다.

설정값이 비어 있으면 기존처럼 현재 브라우저에만 저장됩니다. 발표용 공개 테스트 규칙은 누구나 읽고 쓸 수 있으니, 실제 서비스로 쓰려면 로그인/권한 규칙을 추가해야 합니다.

## 화면 분리

- `index.html`: 사용자 배포용 화면입니다. 추천 요청, 대기 상태, 결과 확인만 제공합니다.
- `admin.html`: 관리자 전용 화면입니다. 요청 확인, AI 프롬프트 복사, 추천 결과 저장, 강의 데이터 관리를 제공합니다.
- 사용자에게는 `index.html` 주소만 공유하고, 관리자는 `admin.html` 주소로 접속하세요.

주의: 정적 호스팅만으로는 관리자 주소를 완전히 보호할 수 없습니다. 실제 외부 배포에서는 Firebase 로그인과 Firestore 보안 규칙을 함께 설정해야 합니다.
